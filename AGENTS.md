| ---------------- | ------------------------------------------------------------------------ |

| `npm run dev`      | Starts the local development server with Hot Module Replacement (HMR).   |
| `npm run build`    | Compiles and bundles the application for production.                     |
| `npm run preview`  | Serves the production build locally for verification.                    |
| `npm run test`     | Runs all unit and component tests via Vitest.                            |
| `npm run test:e2e` | Executes end-to-end tests using Playwright.                              |
| `npm run lint`     | Runs Biome to analyze code for potential errors and style violations.    |
| `npm run format`   | Formats the entire codebase using Biome.                                 |
